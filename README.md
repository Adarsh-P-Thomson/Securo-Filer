# Securo-Protocol
A desktop app to hash and encrypt/decrypt files.

Front End:FLTK
# hashing algorithms:
- SHA-256
- SHA-512
- SHA-3-256
- SHA-3-512
# encryption algorithms:
- AES-256-CBC
- AES-256-GCM
# encryption modes:
- CBC (Cipher Block Chaining)
- GCM (Galois/Counter Mode)
# encryption key size:
- 256 bits (32 bytes)
# encryption key size:
- 128 bits (16 bytes)
